Competências

HTML & CSS
Intermediário

HTML e CSS trabalham juntos para criar páginas da web visualmente atraentes e bem estruturadas. O HTML fornece a base estrutural do conteúdo, enquanto o CSS define como esse conteúdo deve ser exibido aos visitantes, permitindo a criação de layouts atraentes e responsivos.

Javascript
Intermediário

O JavaScript é uma linguagem de programação amplamente usada para tornar as páginas da web interativas e dinâmicas. Ao contrário do HTML e do CSS, que lidam com a estrutura e o estilo de uma página, o JavaScript permite adicionar comportamento e interatividade a ela.

IFAL
2022-2025

O IFAL (Instituto Federal de Educação, Ciência e Tecnologia de Alagoas) é uma instituição de ensino pública federal localizada no estado de Alagoas, Brasil. Fundado em 2008, o IFAL faz parte da rede de Institutos Federais do Brasil, que oferecem educação profissional, tecnológica e superior em diversas áreas.
